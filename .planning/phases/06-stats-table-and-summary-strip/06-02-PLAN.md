**3. Add helper functions:**

`splitHits(totalHits)` -- deterministic mock split:
- `qr = Math.round(totalHits * 0.3)`
- `clicks = totalHits - qr` (guarantees sum = totalHits)
- Returns `{ clicks: clicks, qr: qr }`

`formatCurrency(value)` -- safe currency formatting:
- `var rounded = Math.round(value * 100) / 100;` (snap to cents, kill float drift)
- `var abs = Math.abs(rounded);`
- `var formatted = '$' + abs.toFixed(2);`
- Return `rounded < 0 ? '-' + formatted : formatted;`

`formatDate(dateString)` -- relative date formatting (exact copy from client-links.js lines 889-898):
- If falsy, return '-'
- Parse date, if invalid return raw string
- Calculate days difference from now
- 0 = "Today", 1 = "Yesterday", <7 = "X days ago"
- Older: `date.toLocaleDateString(undefined, { year: 'numeric', month: 'short', day: 'numeric' })`

`formatDateRange(startDate, endDate)` -- for empty state message:
- Format both dates with `toLocaleDateString(undefined, { year: 'numeric', month: 'short', day: 'numeric' })`
- Return `startFormatted + ' to ' + endFormatted`

**4. Add sorting function:**

`getSortedData()`:
- Parse `state.sort` into field and direction
- Shallow copy `state.data` with `.slice()`
- Sort using comparator:
  - For `date` field: compare with `new Date(aVal).getTime()`
  - For numeric fields (totalHits, hitCost, balance): direct numeric comparison
- Return sorted array

`updateSortIndicators()` -- exact same pattern as client-links:
- Parse `state.sort`
- Loop over `.tp-ud-sortable` elements
- Remove `tp-ud-sort-active` class and reset icon to `fa-sort`
- For matching field: add `tp-ud-sort-active`, set icon to `fa-sort-up` or `fa-sort-down`

**5. Add rendering functions:**

`renderTable()`:
- Call `getSortedData()` to get sorted array
- Compute pagination: `totalRecords`, `totalPages`, clamp `currentPage`
- Slice for current page
- Call `renderRows(pageData)`
- Call `renderPagination(totalRecords, totalPages)`
- Call `updateSortIndicators()`
- Show `$tableContainer`, hide `$emptyState`

`renderRows(pageData)`:
- Empty `$tbody`
- For each day in pageData, build a `<tr>` with:
  - Date cell: `<td class="tp-ud-col-date" data-label="Date"><span class="tp-ud-date">` + formatDate(day.date) + `</span></td>`
  - Hits cell: `<td class="tp-ud-col-hits" data-label="Hits">` containing:
    - `<div class="tp-ud-hits-cell">`
    - `<span class="tp-ud-hits-total">` + day.totalHits.toLocaleString() + `</span>`
    - `<span class="tp-ud-hits-breakdown"><i class="fas fa-mouse-pointer"></i> ` + split.clicks.toLocaleString() + ` <i class="fas fa-qrcode ms-1"></i> ` + split.qr.toLocaleString() + `</span>`
  - Cost cell: `<td class="tp-ud-col-cost" data-label="Cost"><span class="tp-ud-cost">` + formatCurrency(day.hitCost) + `</span></td>`
  - Balance cell: `<td class="tp-ud-col-balance" data-label="Balance"><span class="tp-ud-balance">` + formatCurrency(day.balance) + `</span></td>`
- Append each row to `$tbody`

`renderPagination(totalRecords, totalPages)`:
- Empty `$paginationList`
- If `totalPages <= 1`: hide `$pagination`, show record count in info, return
- Show `$pagination`
- Use windowed pagination with maxVisible=5 (exact same algorithm as client-links):
  - Calculate startPage, endPage window
  - Prev button (disabled if page 1)
  - First page + ellipsis if needed
  - Page number buttons (active for current page)
  - Last page + ellipsis if needed
  - Next button (disabled if last page)
- Update info text: "Showing X-Y of Z days"

`renderSummaryCards(data)`:
- If no data or empty: hide `$summaryStrip`, return
- Calculate totals using integer-cent math for cost:
  - `totalHits` = sum of all `day.totalHits`
  - `totalCostCents` = sum of `Math.round(day.hitCost * 100)` for each day
  - `totalCost = totalCostCents / 100`
  - `latestBalance` = last item's balance (data is sorted by date from API, use `data[data.length - 1].balance`)
  - `dailyAvg = data.length > 0 ? Math.round(totalHits / data.length) : 0`
- Build HTML using `buildStatCard(icon, value, label, secondary)` helper:
  - Card 1: icon `fa-chart-line`, value = totalHits.toLocaleString(), label = "Total Hits", secondary = "~" + dailyAvg.toLocaleString() + "/day"
  - Card 2: icon `fa-dollar-sign`, value = formatCurrency(totalCost), label = "Total Cost", secondary = data.length + " days"
  - Card 3: icon `fa-wallet`, value = formatCurrency(latestBalance), label = "Balance", secondary = "Current"
- Set `$summaryStrip.html(html).show()`

`buildStatCard(icon, value, label, secondary)`:
- Returns HTML string for one stat card:
```html
<div class="tp-ud-stat-card">
    <div class="tp-ud-stat-icon"><i class="fas {icon}"></i></div>
    <div class="tp-ud-stat-body">
        <div class="tp-ud-stat-value">{value}</div>
        <div class="tp-ud-stat-label">{label}</div>
        <div class="tp-ud-stat-secondary">{secondary}</div>
    </div>
</div>
```

`showEmptyState()`:

- Set `$emptyRange.text()` to: "No activity from " + formatDateRange(state.dateStart, state.dateEnd)
- Show `$emptyState`

**7. Add new event bindings in `bindEvents()`:**

Sort handler (delegated -- survives DOM re-renders):
```javascript
$(document).on('click', '.tp-ud-sortable', function() {
    var field = $(this).data('sort');
    if (!field) return;

    var parts = state.sort.split(':');
    if (parts[0] === field) {
        state.sort = field + ':' + (parts[1] === 'asc' ? 'desc' : 'asc');
    } else {
        state.sort = field + ':asc';
    }
    state.currentPage = 1;  // Always reset to page 1 on sort change
    renderTable();           // Client-side re-render, no AJAX
});
```

Pagination handler (delegated):
```javascript
$(document).on('click', '#tp-ud-pagination-list .page-link', function(e) {
    e.preventDefault();
    var page = $(this).data('page');
    if (page && page !== state.currentPage && page >= 1) {
        state.currentPage = page;
        renderTable();  // Client-side re-render, no AJAX
    }
});
```

**CRITICAL: Do NOT show skeleton during sort/pagination.** These are client-side operations. Only `loadData()` shows the skeleton.
