**Cell Rendering -- extend renderRows():**

1. Add a helper function `buildOtherServicesCell(day)` that:
   - Checks `day.otherServices` for null/undefined -- if null or amount is 0, returns a `<td>` with `<span class="tp-ud-other-zero">$0.00</span>`
   - If `otherServices.amount > 0`, returns a `<td>` with `<span class="tp-ud-other-amount">` showing `+` prefix + `formatCurrency(os.amount)`, with `data-bs-toggle="tooltip"`, `data-bs-html="true"`, and `data-bs-title` containing the tooltip HTML
   - The `<td>` must have `class="tp-ud-col-other"` and `data-label="Other Services"`

2. Add a helper function `buildTooltipContent(items)` that:
   - For single item (items.length === 1): returns just the escaped description text
   - For multiple items: returns each as `"Description (+$amount)"` joined by `<br>`
   - Uses `escapeHtml()` on all description text before insertion

3. Add an `escapeHtml(str)` utility function that replaces `&`, `<`, `>`, `"`, `'` with HTML entities. Also add `escapeAttr(str)` that escapes for use inside HTML attribute values (double-quote encoding).

4. In `renderRows()`, insert the new cell call between the Hits `<td>` and Cost `<td>` in the row HTML string.

**Tooltip Lifecycle:**

5. Add a `disposeTooltips()` function that finds all `[data-bs-toggle="tooltip"]` elements inside `$tbody`, gets the `bootstrap.Tooltip` instance via `bootstrap.Tooltip.getInstance(el)`, and calls `dispose()` if it exists. Call this BEFORE `$tbody.empty()` in `renderRows()`.

6. Add an `initTooltips()` function that queries `#tp-ud-tbody [data-bs-toggle="tooltip"]` and creates `new bootstrap.Tooltip(el, { trigger: 'hover focus', placement: 'top', container: 'body' })` for each. Call this AFTER rows are appended in `renderRows()` (or in `renderTable()` after `renderRows()` completes).

**Sorting:**

7. In the sort comparator function (`getSortedData()` or equivalent), add handling for the `otherServices` sort field. When the sort field is `otherServices`, extract the numeric value as: `(day.otherServices && day.otherServices.amount) || 0`. Compare these numeric values for ascending/descending sort.

**Summary Card:**

8. In `renderSummaryCards()` (or equivalent), add computation of Other Services total:
   - Iterate all data records (not just the current page)
   - Sum `otherServices.amount` for records where `otherServices !== null && otherServices.amount > 0`
   - Use integer-cents accumulation pattern: `Math.round(amount * 100)` then divide by 100 at end
   - Count days with credits for the secondary text
   - Call `buildStatCard('fa-hand-holding-dollar', '+' + formatCurrency(total), 'Other Services', N + ' day(s) with credits')`
   - Place this card 4th (after the existing 3 cards, at the end of the strip)

**Null safety throughout:**
- Every access to `day.otherServices` must check for null first
- When wallet API fails, ALL records have `otherServices: null` -- column shows all $0.00, summary shows +$0.00, 0 days with credits
